React Native (简称RN)是Facebook于2015年4月开源的跨平台移动应用开发框架

目前支持iOS和安卓两大平台

RN使用Javascript语言，类似于HTML的JSX，以及CSS来开发移动应用

。。。。。。

优点

  1. 对应到前端开发，整个系统结构是这样
    * JSX vs HTML
    * CSS-layout vs css
    * ECMAScript 6
    * React native View vs DOM
  2. 两端并行开发，RN可以调用android和ios原生代码实现需要的功能
  3. 热更新
  4. 社区力量，三方组件

缺点

  1. 初期学习成本高，需要学习ES6，函数式编程思想，新的标签，新的控件，开发环境的配置
  2. RN本身更新很快，现在到0.47版本，这就意味着不稳定，坑多
  3. 第三方组件之间会有冲突，或者组件本身有坑
  4. 可以预见的性能问题，比如list列表
